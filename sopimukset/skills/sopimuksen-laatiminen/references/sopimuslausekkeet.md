# Sopimuslausekkeet — kommentoidut mallit

Mallilausekkeita yleisimpiin sopimuksen kohtiin. **Nämä ovat lähtökohtia, eivät valmiita
ratkaisuja** — sovita sopimustyyppiin, osapuolten asemaan ja riskeihin. Tarkista
lakiviittaukset `oikeustutkimus`-skillillä. Hakasulkeet `[ ]` merkitsevät täydennettävää.

## Osapuolet

```
Tämän sopimuksen ovat tehneet:

[Yrityksen virallinen nimi] (y-tunnus [0000000-0]), jäljempänä "Toimittaja", ja
[Yrityksen virallinen nimi] (y-tunnus [0000000-0]), jäljempänä "Tilaaja",

jäljempänä yhdessä "Osapuolet" ja erikseen "Osapuoli".
```

## Sopimuksen kohde

```
Toimittaja toimittaa Tilaajalle [palvelun/tavaran kuvaus] tämän sopimuksen ja sen
liitteiden mukaisesti. Toimituksen laajuus ja vaatimukset on määritelty liitteessä [1].
```

## Hinta ja maksuehdot

```
Sopimuksen hinta on [summa] euroa (alv 0 %). Arvonlisävero lisätään kulloinkin
voimassa olevan verokannan mukaisesti. Toimittaja laskuttaa [laskutusperuste].
Maksuaika on [14] päivää laskun päiväyksestä. Viivästyskorko määräytyy korkolain
mukaan.
```
*Huom: tarkista viivästyskoron peruste korkolaista `oikeustutkimus`-skillillä.*

## Salassapito

```
Osapuoli ei saa ilmaista kolmannelle eikä käyttää muuhun kuin tämän sopimuksen
mukaiseen tarkoitukseen toiselta Osapuolelta saamiaan luottamuksellisia tietoja.
Salassapitovelvollisuus ei koske tietoa, joka on yleisesti saatavilla, jonka Osapuoli
on saanut oikeutetusti kolmannelta ilman salassapitovelvoitetta tai jonka Osapuoli on
itsenäisesti kehittänyt. Salassapitovelvollisuus on voimassa [5] vuotta sopimuksen
päättymisestä.
```

## Vastuu ja vastuunrajoitus

```
Osapuoli vastaa toiselle Osapuolelle sopimusrikkomuksella aiheuttamastaan välittömästä
vahingosta. Osapuoli ei vastaa välillisistä vahingoista, kuten saamatta jääneestä
voitosta tai tuotannonmenetyksestä. Osapuolen kokonaisvastuu tämän sopimuksen
perusteella rajoittuu [sopimuksen arvoon / summaan] euroon.

Vastuunrajoitukset eivät koske vahinkoa, jonka Osapuoli on aiheuttanut tahallaan tai
törkeällä huolimattomuudella, eivätkä salassapitovelvollisuuden rikkomista.
```
*Huom: vastuun rajoittaminen tahallisuuden/törkeän huolimattomuuden osalta ei ole
pätevää — pidä poikkeus mukana.*

## Sopimussakko

```
Mikäli Osapuoli rikkoo [yksilöity velvoite, esim. salassapitovelvollisuutta], se on
velvollinen suorittamaan toiselle Osapuolelle sopimussakkona [summa] euroa kutakin
rikkomusta kohden. Sopimussakon lisäksi voidaan vaatia korvausta sopimussakon ylittävältä
osalta todennetusta vahingosta.
```
*Huom: kohtuuton sopimussakko voi altistua sovittelulle (oikeustoimilain 228/1929 36 §).*

## Ylivoimainen este (force majeure)

```
Osapuoli vapautuu velvoitteistaan ja vastuusta, jos sopimuksen täyttäminen estyy tai
viivästyy Osapuolen vaikutusmahdollisuuksien ulkopuolisesta esteestä, jota Osapuolen ei
kohtuudella voitu edellyttää ottaneen huomioon sopimusta tehtäessä (esim. luonnonmullistus,
viranomaismääräys, yleinen liikenteen tai energiansaannin keskeytys). Osapuolen on
viipymättä ilmoitettava esteestä toiselle Osapuolelle.
```

## Voimassaolo ja irtisanominen

```
Sopimus tulee voimaan, kun molemmat Osapuolet ovat sen allekirjoittaneet, ja on voimassa
[toistaiseksi / määräajan [pp.kk.vvvv] saakka]. Kumpikin Osapuoli voi irtisanoa sopimuksen
[3] kuukauden irtisanomisajalla. Osapuoli voi purkaa sopimuksen päättymään välittömästi,
jos toinen Osapuoli rikkoo sopimusta olennaisesti eikä korjaa rikkomusta [30] päivän
kuluessa kirjallisesta huomautuksesta.
```

## Sovellettava laki ja riidanratkaisu

```
Tähän sopimukseen sovelletaan Suomen lakia.

[Vaihtoehto A — yleinen tuomioistuin:]
Tästä sopimuksesta aiheutuvat erimielisyydet ratkaistaan [paikkakunnan] käräjäoikeudessa.

[Vaihtoehto B — välimiesmenettely:]
Tästä sopimuksesta aiheutuvat erimielisyydet ratkaistaan lopullisesti välimiesmenettelyssä
Keskuskauppakamarin välimiesmenettelysääntöjen mukaisesti. Välimiesoikeuden paikka on
[Helsinki] ja menettelyn kieli suomi.
```

## Muut ehdot (sekalaiset)

```
Sopimuksen muutokset on tehtävä kirjallisesti ja molempien Osapuolten allekirjoituksin.
Osapuoli ei saa siirtää sopimusta kolmannelle ilman toisen Osapuolen kirjallista
suostumusta. Sopimuksen liitteet ovat sen erottamaton osa; ristiriitatilanteessa sopimusteksti
on liitteisiin nähden etusijalla. Tämä sopimus korvaa Osapuolten aiemmat samaa asiaa
koskevat sopimukset ja sopimisen.
```
